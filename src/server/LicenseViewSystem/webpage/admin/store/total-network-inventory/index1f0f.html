MSP/ITSP licensing

If you are an MSP/ITSP (Managed/IT Services Provider), you can use this license to inventory the computers of your clients and customers.

What is a node?

A node is a computer, server, network printer, router, or any other network device with an IP address.

...

MSP/ITSP license allows you to inventory specified number of nodes for each customer.

While using the program, you may also add custom assets to your storage manually. These are NOT counted as nodes, so you can have any number of them.
